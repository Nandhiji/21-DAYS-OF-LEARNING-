CONSTRUCTOR:
 A CONSTRUCTOR is a special method inside  a class .
 -->> it runs automatically when an object is created.
 -->> used to initialize variables.

 IMPORTANT RULES:

1. constructor name must be same as class name.
2.it has NO return type  (even void ).
3.it runs automatically once the object is created.

TYPES OF CONSTRUCTOR:
1.DEFAULT CONSTRUCTOR
2.PARAMETERIZED CONSTRUCTOR

1.DEFAULT CONSTRUCTOR:
 
 constructor(){
    System.out.println("hi");
 }

 2.PARAMETERIZED CONSTRUCTOR;

 constructor(String n, int a){
   name=n;
   age=a;      // used to pass values during object creation.

 }
