static keyword in brief :
1️⃣ What is static in Java?

Simple definition:

static means the variable or method belongs to the CLASS
not to the OBJECT

Normal variables belong to objects.
static variables belong to the class itself.

⚡ Super Important Rule

Inside a static method you cannot directly use non-static variables.
